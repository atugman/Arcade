Andrew's Arcade

About this Project

Creating Andrew's Arcade was an educational exercise designed to practice using: JavaScript, HTML5 Canvas, Express, Node.js, Mongo and Mongoose.

I used HTML5 Canvas and JavaScript to create the game.

I used Express, Node.js, Mongo and Mongoose to create the login, high score, and user creation capabilities.

Users' scores are stored in a database using MongoDB.
